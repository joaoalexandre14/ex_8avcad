
# Exercise 8 - Species Richness Regression Analysis

## Overview
This repository contains the analysis for Exercise 7, focusing on the `EFIplus_medit` dataset. The objective is to explore the relationship between species richness and several continuous environmental variables through Simple and Multiple Linear Regressions.

## Workflow
1. **Univariate Analysis:** Simple linear regressions applied to each continuous environmental variable, including data transformation assessments.
2. **Multiple Linear Regression:** Evaluating combined effects and comparing with previous isolated responses using Partial Dependence Plots.
3. **Parsimonious Model:** Assessing multicollinearity through Variance Inflation Factor (VIF) to build a stable, simplified regression model.

## Files in this Repository
* `ex8avcad.ipynb`: Main script containing data processing, modeling, and visualization.
* `EFIplus_medit.csv`: The dataset used for the analysis.

## Required Libraries
To run the code, ensure the following Python libraries are installed:
* `pandas`
* `numpy`
* `statsmodels`
* `matplotlib`
* `seaborn`
